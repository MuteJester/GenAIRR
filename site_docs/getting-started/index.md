# Getting Started

<p class="lead">The fastest path from <code>pip install GenAIRR</code>
to a working simulation, organised as a four-page sequence.</p>

## The four pages, in order

1. **[Installation](installation.md)** - wheels, Python 3.9+, no
   Rust toolchain needed.
2. **[Quick start](quick-start.md)** - your first simulation in
   under 5 minutes. 100 productive heavy chains, AIRR records
   returned.
3. **[Your first AIRR record](first-airr-record.md)** -
   field-by-field tour of what the engine emits per record.
4. **[Export the results](export-results.md)** - write TSV /
   CSV / FASTA / FASTQ / pandas DataFrame.

After this section, the natural next stop is
**[The Experiment builder](../guides/experiment-builder.md)** -
the DSL surface you'll use for every non-trivial simulation.

For a higher-level routing by intent, see
**[Choose your path](../learn.md)**.

# GenAIRR Documentation

**GenAIRR** is a high-performance simulator for adaptive immune receptor sequences (BCR and TCR).
It produces synthetic sequences in AIRR format with **absolute ground-truth annotations** — every
V/D/J call, every mutation position, every trimming length is known exactly because the simulator
constructed the sequence from scratch.

```python
from GenAIRR import Experiment
from GenAIRR.ops import rate

result = Experiment.on("human_igh").mutate(rate(0.02, 0.08)).run(n=1000, seed=42)
```

## Why GenAIRR?

- **Ground truth** — Every output field reflects what actually happened during simulation, not what an aligner inferred. This makes GenAIRR data ideal for training and benchmarking alignment tools.
- **Fast** — A C simulation engine produces 15,000–30,000 sequences per second on a single core.
- **Broad coverage** — 106 built-in configurations across 23 species, sourced from IMGT and OGRDB.
- **Realistic** — Context-aware S5F somatic hypermutation, D-gene inversion, receptor revision, class switch recombination, sequencing artifacts, and more.
- **Zero mandatory dependencies** — Pure Python + compiled C extension. Pre-built wheels for Linux, macOS, and Windows.

## The Experiment DSL

GenAIRR uses a fluent builder called `Experiment` with five biological phases:

```python
from GenAIRR import Experiment
from GenAIRR.ops import (
    rate, model,
    with_5prime_loss, with_3prime_loss,
    with_indels, with_ns,
)

result = (
    Experiment.on("human_igh")

    # Somatic hypermutation
    .mutate(
        model("s5f"),
        rate(0.02, 0.08),
    )

    # Sequencing artifacts
    .sequence(
        with_5prime_loss(min_remove=5, max_remove=30),
        with_3prime_loss(min_remove=5, max_remove=20),
    )

    # Post-sequencing noise
    .observe(
        with_indels(prob=0.01),
        with_ns(prob=0.005),
    )

    .run(n=1000, seed=42)
)
```

Each phase maps to a stage of a real sequencing experiment. You only include the phases you need — calling `.run()` on a bare `Experiment.on(...)` produces unmutated, uncorrupted rearrangements.
